## Reglas de la presentación

* La presentación es individual
* Pueden utilizar, para hacer su código, cualquier 
  herramienta de IA.
* PAra verificar si es su código se le pedirá que
  puedan realizarlo a mano
* Los prácticos tienen logs (Log4j). El práctico que no
  tenga logs tiene 0.

Niveles de tipos de errores:

DEBUG / INFO / WARN / ERROR / FATAL
